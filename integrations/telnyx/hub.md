# Telnyx integration

## Description

The Telnyx integration enables seamless communication between your AI-powered chatbot and Telnyx, a powerful cloud communications platform. Connect your chatbot to Telnyx and leverage programmable voice and SMS capabilities. With this integration, you can automate interactions, send personalized notifications, handle inquiries, and provide customer support directly through Telnyx.

## Features

- Send and receive SMS messages
- Handle voice call events (initiated, answered, hangup, failed)
- Proactive conversation initiation
- User identification via phone numbers

## Setup

1. Obtain your Telnyx API key from the Telnyx portal
2. Configure your Call Control Application ID for voice calls (optional)
3. Configure your Messaging Profile ID for SMS (optional)
4. Set up webhooks to receive incoming messages and call events

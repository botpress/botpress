# CAMB AI Integration

This integration allows your bot to use [CAMB AI](https://camb.ai/) for text-to-speech synthesis, text translation, translated TTS, and voice cloning across 140+ languages.

CAMB AI's MARS models provide ultra-low-latency speech synthesis (~150ms) with natural-sounding voices.

## Features

- **Text-to-Speech**: Generate natural speech from text using MARS models
- **Translation**: Translate text between 140+ languages
- **Translated TTS**: Translate text and generate speech in the target language
- **Voice Cloning**: Clone voices from audio samples
- **Voice Listing**: Browse available voices
